# Plant Sales Analysis

## Short Description
This project analyzes plant sales data to understand sales distribution across different plant sizes and identify patterns that can help improve inventory planning and business decisions. The analysis is performed using spreadsheet-based data and basic data analysis techniques.

## Introduction
Plant businesses often manage different plant sizes such as small, medium, and large. Understanding which sizes sell the most helps optimize inventory, pricing strategies, and marketing efforts.

This project performs exploratory data analysis on plant sales data to uncover useful insights about plant size demand and sales distribution.

---

## Dataset / Inputs
The dataset used in this project:

- **File:** `plant_sales.xlsx`
- **Description:** Contains plant sales data categorized by plant size.
- **Key Fields:**
  - `Plant Size`
  - `Sales Count / Quantity`
